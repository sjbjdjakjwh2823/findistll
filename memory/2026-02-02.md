## Spoke E 설계 완료: "The Oracle Layer"

### 핵심 역할
온톨로지 그래프(Spoke D)의 **미래 상태 예측**
- 예: "금리 0.25% 인상 시 3개월 후 기술주-에너지주 연결 강도(β) 변화 예측"

### 통합 데이터 스키마
| 구성 요소 | 출처 | 역할 |
|-----------|------|------|
| Contextual Fact | Spoke C | 정규화된 재무/매크로 데이터 |
| Relational Weights | Spoke D | 노드 간 연결 강도 (β) |
| Confidence Vector | Spoke B | 신뢰도 점수 |
| Reasoning Path | Spoke A | CoT 로그 |
| Outcome Label | Market | 실제 시장 반응 (피드백) |

### 예측 신뢰도 가중치 공식
```
Wpred = Σ(Ci · βi · ΔMi)
```
- Ci: Spoke B 신뢰도
- βi: Spoke D 민감도
- ΔMi: 매크로 변수 변화량

### 3단계 구현 가이드
1. **온톨로지 스냅샷 아카이빙**: 시점별 그래프 상태 저장
2. **피드백 루프**: 예측→관측→오차분석→가중치 보정
3. **통합 사고 엔진**: FinRobot Brain에 Spoke E 연결

### Palantir 핵심 가치
- Self-Evolving Ontology: 데이터 축적 → 그래프 자동 정교화
- Scenario Grounding: If-Then 시나리오의 확률적 미래 제시
- PhD Research Asset: 학술적 + 상업적 가치
